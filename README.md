## âš™ï¸ Configuration (`config.yaml`)

Customize your report generation through this file:

```yaml
output_folder: ../rkd-backtest-core/backtest
output_report: report
timeframes:
  - 15m
  - 30m
  - 1h
  - 4h
start_year: 2020
end_year: 2024
```
